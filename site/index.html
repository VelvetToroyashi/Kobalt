Kobalt.
Private, Powerful, Proficient.

Kobalt is a privacy-first Discord bot with robust tools to empower moderators and end users alike.

Privacy Means Privacy.
Kobalt means privacy first. All stored data is opt-in, period.
Open Source
Kobalt is an open-source project! You're welcome to audit the source, open PRs, or even host your own instance of Kobalt.
Robust Configuration
Different servers have different needs. From VC limits to repeat infractions, you can configure just about everything.
Why Kobalt?

Kobalt is not just any run-of-the-mill multipurpose bot. From convenient and industry-standard features to niche and alluring features you didn't know you need, Kobalt has something for everyone.

Localize Times

It's common to speak with people on the other side of the globe, which means dealing with timezones, and having to ask "what time is it for you?"

Kobalt saves you from asking this question with a simple right click.

I'm available between 4PM and 10PM! I see the following times in that message:
4PM ➜ 9:00 AM
10PM ➜ 3:00 PM
Artist Verification

Verifying artists used to be a tedious process of looking up someone's twitter,
looking through their posts, reverse-searching, and making a judgement call.

Kobalt improves on this vastly by not only keeping a database of known good/bad artists, but introduces innovating tooling to let artists automatically verify themselves.

The images submitted most likely belong to this artist. Platforms: @Example_Artist @example.artist ~ExampleArtist This artist has been verified since June 12th, 2023.
Anti-Raid that works.

Kobalt is not a dedicated anti-raid bot; that'd be Beemo.

Kobalt offers a set-it-and-forget-it anti-raid that's trivial to set up.
Unlike other bots that use a floodgate technique to stop raids, Kobalt uses an algorithm that checks multiple factors of each user to determine bad actors, without hindering genuine users.

Extensible plugin system.

Kobalt offers a lot of features aside from what's listed here. Need something niche and specific to your server? Just make a plugin!
Just a few lines of code, and you're off to the races. 

            
exports.commands = [
  {
    name: "my_plugin_command",
    description: "A pretty cool command from a plugin!",
    parameters: {
      string: CommandHelper.string(min: 10),
      user: CommandHelper.user(),
      delay: CommandHelper.delay(),

    }
  }
]

async function my_plugin_command(state, args) {
  await args.user.send_message("Hi from plugin!!");
}
